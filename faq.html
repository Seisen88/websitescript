Frequently Asked Questions

Find answers to common questions about Seisen

What is Seisen?

Seisen is a premium platform that provides high-quality scripts and a powerful Lua obfuscator. We offer both free and premium access to our script library, along with advanced obfuscation tools powered by Prometheus.

Is Seisen safe to use?

Yes, Seisen is completely safe. We don't collect personal information, and all our scripts are tested for security. Our obfuscator runs locally in your browser or on our secure backend server with Prometheus.

How do I get started?

Getting started is easy! Simply obtain a free or premium access key, browse our script library, and copy the scripts you need. For the obfuscator, just paste your Lua code and click obfuscate.

What types of scripts do you offer?

We offer a variety of scripts including game scripts, utility scripts, and automation tools. Premium members get access to exclusive advanced scripts with additional features and regular updates.

How often are scripts updated?

Our scripts are regularly updated to ensure compatibility and add new features. Premium scripts receive priority updates, and we notify users through Discord when major updates are released.

Can I request custom scripts?

Premium members can request custom scripts or modifications through our Discord server. We review all requests and implement the most popular ones in our updates.

What is the Lua Obfuscator?

Our Lua Obfuscator is a powerful tool that transforms your Lua code to make it harder to reverse-engineer. It's powered by Prometheus, supporting both Lua 5.1 and LuaU (Roblox Lua), with multiple obfuscation presets from minify to strong.

Is the obfuscator free to use?

Yes! Our obfuscator is completely free to use with no limitations. You can obfuscate as much code as you want using any of the available presets.

What obfuscation presets are available?

We offer four presets: Minify (basic compression), Weak (light obfuscation with variable renaming), Medium (moderate obfuscation with control flow changes), and Strong (heavy multi-layer obfuscation).

Does obfuscation affect performance?

Higher obfuscation levels may slightly impact performance due to added complexity. The Minify preset actually improves performance by removing whitespace. Medium and Strong presets add some overhead but maintain functionality.

Can I upload files to obfuscate?

Yes! You can upload .lua files directly using the file upload button. The code will be loaded into the editor, and you can then obfuscate it normally.

What's the difference between Free and Premium?

Free tier provides access to basic scripts with a time-based key system that requires periodic renewal. Premium offers unlimited access to all scripts, priority updates, custom script requests, and no time limitations.

How much does Premium cost?

Pricing information is available on our Premium page or through our Discord server. We offer various plans to suit different needs and budgets.

How do I upgrade to Premium?

Visit our Premium page and click "Upgrade Now", or join our Discord server for direct assistance with the upgrade process and payment options.

Can I cancel my Premium subscription?

Yes, you can cancel anytime. Contact us through Discord to manage your subscription. You'll retain access until the end of your billing period.

What Lua versions are supported?

Our obfuscator supports Lua 5.1 and LuaU (Roblox Lua). You can select your Lua version in the obfuscator settings before processing your code.

Why isn't my code obfuscating?

Check for syntax errors in your code first. Make sure the backend server is running if you're using the API mode. If issues persist, try a different obfuscation preset or contact us on Discord with the error message.

Is my code stored on your servers?

No, we don't store your code. When using the obfuscator, your code is processed and immediately discarded. History is stored locally in your browser only.

Can I use Seisen offline?

The obfuscator requires a backend connection to use Prometheus. However, you can download and run the backend locally by following the setup instructions in GitHub repository.

What browsers are supported?

Seisen works on all modern browsers including Chrome, Firefox, Edge, and Safari. We recommend using the latest version for the best experience.
